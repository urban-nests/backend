## Description

Urban Nest backend

## Version

- Spring Boot 3.2.5
- JDK 17

## Installation

## Running the app

```bash
./gradlew bootRun
nohup java -jar backend-0.0.1.SNAPSHOT.jar
```

## Build

```bash
FatBuild:
./gradlew clean bootJar
```

## Test

```bash
./gradlew test
```
